Guía para instalar y empezar con Java

Introducción

Bienvenido a nuestra guía para comenzar a programar en Java. En esta guía, te proporcionaremos instrucciones paso a paso para instalar Java en tu sistema y te daremos los primeros pasos para comenzar a escribir y ejecutar código Java.

Instalación de Java

Paso 1: Descargar Java Development Kit (JDK)

  1. Abre tu navegador web y visita el sitio web oficial de Oracle en oracle.com.
  2. En la sección "Java SE Development Kit", elige la versión más reciente de JDK para tu sistema operativo y haz clic en el enlace de descarga correspondiente.
  3. Acepta los términos y condiciones y selecciona la descarga adecuada para tu sistema operativo (Windows, Linux o macOS).
  4. Espera a que se complete la descarga.

Paso 2: Instalar JDK

  1. Ejecuta el archivo de instalación que descargaste anteriormente.
  2. Sigue las instrucciones en pantalla y completa la instalación.
  3. Asegúrate de configurar las variables de entorno del sistema para que Java se pueda ejecutar desde cualquier ubicación en tu sistema.

¡Felicidades! Has instalado Java Development Kit en tu sistema.

Primeros pasos con Java

Paso 1: Configurar el entorno de desarrollo

Para comenzar a programar en Java, necesitarás un entorno de desarrollo integrado (IDE). Hay varios IDE populares disponibles para Java, como Eclipse, IntelliJ IDEA y NetBeans. Elige el que más te guste y descárgalo desde su sitio web oficial.

Paso 2: Escribir y ejecutar código Java

Una vez que hayas configurado tu IDE, estás listo para escribir y ejecutar tu primer programa en Java. Aquí tienes un ejemplo sencillo de un programa "Hola, mundo":

public class HolaMundo {
  public static void main(String[] args) {
    System.out.println("¡Hola, mundo!");
  }
}

Copia este código en tu IDE, guarda el archivo con el nombre "HolaMundo.java" y ejecútalo. Verás la salida "¡Hola, mundo!" en la consola de tu IDE.

Ejercicio 1: Saludo personalizado

Ahora es tu turno. Modifica el código anterior para que imprima un saludo personalizado con tu nombre. Por ejemplo:

public class HolaPersonalizado {
  public static void main(String[] args) {
    String nombre = "Tu nombre";
    System.out.println("¡Hola, " + nombre + "!");
  }
}

Cambia `"Tu nombre"` por tu nombre real, guarda los cambios y ejecuta el programa nuevamente para ver el saludo personalizado.
