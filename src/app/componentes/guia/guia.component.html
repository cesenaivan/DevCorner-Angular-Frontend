Guía para instalar y empezar con Python

Introducción

Bienvenido a nuestra guía para instalar Python y comenzar a programar. En esta guía, te proporcionaremos instrucciones paso a paso para instalar Python en tu sistema y te daremos los primeros pasos para comenzar a escribir y ejecutar código.

Instalación de Python

Paso 1: Descargar Python

  1. Abre tu navegador web y visita el sitio web oficial de Python en python.org.
  2. Ve a la sección "Downloads" (Descargas) y elige la última versión estable de Python para tu sistema operativo.
  3. Haz clic en el enlace de descarga y espera a que se complete la descarga.

Paso 2: Instalar Python

  1. Ejecuta el archivo de instalación que descargaste anteriormente.
  2. Asegúrate de marcar la casilla "Add Python to PATH" (Agregar Python al PATH) durante la instalación. Esto te permitirá ejecutar Python desde cualquier ubicación en tu sistema.
  3. Sigue las instrucciones en pantalla y completa la instalación.

¡Felicidades! Has instalado Python en tu sistema.

Primeros pasos con Python

Paso 1: Abrir el intérprete de Python

  1. Abre una terminal o línea de comandos en tu sistema.
  2. Escribe python o python3 y presiona Enter. Esto abrirá el intérprete de Python.

Paso 2: Escribir y ejecutar código

Ahora estás listo para escribir y ejecutar tu primer programa en Python. En el intérprete de Python, escribe el siguiente código:

print("¡Hola, mundo!")

Presiona Enter y verás la salida "¡Hola, mundo!" en la siguiente línea. ¡Felicidades! Acabas de escribir y ejecutar tu primer programa en Python.

Ejercicio 1: Saludo personalizado

Ahora es tu turno. Modifica el código anterior para que imprima un saludo personalizado con tu nombre. Por ejemplo:

print("¡Hola, [Tu nombre]!")

Guarda los cambios y ejecuta el programa nuevamente para ver el saludo personalizado.
